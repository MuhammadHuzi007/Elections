# Interactive Menu Guide

## Overview

The program now features an **interactive menu-driven interface** that allows you to:

- Select any country and year
- Choose specific analysis types
- Compare elections
- View party trends
- Navigate through options easily

## How to Use

### Starting the Program

```powershell
.\bin\election_analysis.exe
```

### Main Menu Options

1. **Analyze Specific Election**
   - Select a country (Jordan or Vanuatu)
   - Select a year (2016, 2020, or 2022)
   - Choose what analysis to view

2. **Compare Two Elections**
   - Select a country
   - Choose two different years
   - View comparative analysis

3. **View Party Trend Analysis**
   - Select a country
   - Select a party
   - View trend across all available years

4. **View Complexity Analysis**
   - See time and space complexity documentation

5. **Exit**
   - Close the program

### Election Analysis Menu

When analyzing a specific election, you can choose:

1. **View Election Statistics**
   - Total votes, seats, candidates, constituencies

2. **View Party-wise Statistics**
   - Vote shares, seats won, candidates per party

3. **View Seat Distribution**
   - Seats won by each party

4. **View Top Candidates**
   - Top N candidates by votes (you specify N)

5. **View Winning Candidates**
   - All elected candidates

6. **View All Analysis**
   - Complete analysis in one view

7. **Back to Main Menu**
   - Return to main menu

## Navigation

- **Enter numbers** to make selections
- **Press Enter** after viewing results to continue
- **Menu clears** between selections for better readability
- **Invalid inputs** are handled gracefully

## Example Workflow

1. Run the program
2. Select option `1` (Analyze Specific Election)
3. Select `1` for Jordan (or `2` for Vanuatu)
4. Select `1` for 2016 (or other available year)
5. Choose analysis type (e.g., `2` for Party Statistics)
6. Press Enter to continue
7. Select `0` to go back to main menu
8. Select `0` again to exit
